Variables de Referencia de Plantilla

permite referenciar un elemento de la plantilla. Este elemento es tipicamente un elemento del DOM o un componente

Ejemplo - declarando una variable de plantilla

ViewChild

con ViewChild se puede hacer referencia a los elementos de la plantilla desde la clase del componente. hay dos formass de hacerlo:

  1. utilizando una variable de referencia de plantilla
  2. utilizando el tipo de componente

Ejemplo - utilizando variable de plantilla

ViewChidren

con ViewChildren se puede agarrar un conjunto de instancias de un componente de una plantilla. Esta referencia es dinamica.

viewChildren tambien permite recorrer los componentes y ademas seleccionar un componente en particular del conjunto
