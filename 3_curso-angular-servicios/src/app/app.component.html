Servicios en Angular

En Angular un servicio es una clase con un proposito bien definido. Los componentes pueden utilizar servicios y asi mismo los serrvicios pueden utilizar otros servicios

  • ventajas: Permite tener codigo organizado y reutilizable
  • crear un servicio con CLI: ng generate service [nombrecarpeta]/[nombredelservicio]
  • ejemplo: ng generate service services/primer_servicio

Inyeccion de Dependencias

Cuando una clase A depende de una clase B se dice que la clase B es dependencia de la clase A. La inyeccion de dependencias es un mecanismo por el cual se inyecta las dependencias de una clase

Ejemplo

servicio HttpClient

Es el standar en Angular para realizar peticiones http. Para hacer uso de este modulo debe ser importado en el app.module

Metodos HTTP

  • HTTP GET: Sirve para leer informacion del servidor
  • HTTP POST: Sirve para enviar informacion al servidor
  • HTTP PUT: sirve para realizar actualizaciones completas de un recurso
  • HTTP PATCH: sirve para realizar actualizaciones parciales de un recurso (ayuda a optimizar la aplicacion)
  • HTTP DELETE: sirve para borrar recursos
